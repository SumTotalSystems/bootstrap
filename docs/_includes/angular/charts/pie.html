Pie

A Pie, a Donut, both are tasty!

The sumt-pie-chart directive can be used to bake a pie or a donut. By default the directive will create a donut chart. The inner radius is set to be 87%. This can be adjusted through the inner-radius attribute. The smaller the percentage the thicker the donut will appear. A value of 0 passed in will cause the chart to be drawn as a pie.
