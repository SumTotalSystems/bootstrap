Left-hand Navigation

A directive to setup a standard page layout.

The sumt-leftnav directive sets up the basic page layout for a page with left-hand navigation.

The HTML below gives an example of how to structure the basic page.

A demo of the left hand navigation can be seen on the Standard Layout #1 and Standard Layout #2 pages.

{% highlight html %}
  1. Home
  2. Previous Page
  3. Current Page

Page TitleSubtitle

Directions and instructions.

Use this document as a way to quickly start any new project.
All you get is this text and a mostly barebones HTML document.

{% endhighlight %}

To setup the left-hand navigation a JSON object will need to be passed into the directive.

{% highlight js %} [{ "title": "Heading1", "id": "Heading1", "children": [ { "title": "subHeading1", "id": "subHeading1", "children": [] }, { "title": "subHeading2", "id": "subHeading2", "children": [] } ] }, { "title": "Heading2", "id": "Heading2", "children": [ { "title": "subHeading3", "id": "subHeading3", "children": [] }, { "title": "subHeading4", "id": "subHeading4", "children": [] }] }]; {% endhighlight %}
