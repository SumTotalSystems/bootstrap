Drag and Drop Lists

Example

Drag and drop the list items to move them around. These lists and containers may be styled as desired. A <ul> is highly recommended for the host element of the dnd-list directive, but any other kind of element may be placed within, not just <li>'s. You may override css attributes such as the ul[dnd-list] .dndPlaceholder and ul[dnd-list] .dndDraggingSource selectors. Watch how the two-way data binding causes the generated JSON object model to update as the list is reordered.

Usage

Pass in a default template using HTML format to the editor using a controller with the following form.

{% highlight js %} .controller('DragDropCtrl', ['$scope', function($scope) { $scope.models = { selected: null, lists: {"A": [], "B": []} }; // Generate initial model for (var i = 1; i <= 3; ++i) { $scope.models.lists.A.push({label: "Item A" + i}); $scope.models.lists.B.push({label: "Item B" + i}); } $scope.$watch('models', function(model) { $scope.modelAsJson = angular.toJson(model, true); }, true); }]); {% endhighlight %}
