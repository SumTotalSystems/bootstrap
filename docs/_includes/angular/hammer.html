Hammer.js

Hammer.js Example

Hammer.js is a mobile friendly web framework that allows touch gestures to be added to any webpage. Any function may be called on event listeners for swipes, taps, presses, pans, pinches, rotates, drag and drops, and more. For more information, visit HammerJS on Github.

The grey box below will detect several kinds of mobile gestures (pans in all directions, taps, presses, pinches, and rotates). Give it a try!

Angular Hammer

To avoid having to mix regular Javascript with angular, there is an angular wrapper for Hammer.js by Ryan Mullins. There are over 30 directives that can be used, each of which can be wired to a custom function in the controller (or all to the same function, as in the example below). Certain events can overlap and should not be used together to prevent unexpected results (ie, a panleft and a pan). Custom gesture recongnizers can also be created (for instance, a two finger pan, a doubletap, etc). For more information and a full list of directives, visit his page on Github.

Usage

The above example is quite trivial - it merely displays the resulting gesture. Of course, you can link the gesture detection to any function you'd normally put in the controller.

{% highlight js %} .controller('hammerCtrl', ['$scope', function($scope) { $scope.option = "No events yet."; $scope.onHammer = function onHammer( val ) { $scope.option = val.type + " gesture detected."; } }]); {% endhighlight %}
