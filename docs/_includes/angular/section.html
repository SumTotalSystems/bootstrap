Section

Sometimes you need to put your DOM in a box. For those situations, try the sumt-section component.

Basic example

The basic panel takes a heading attribute. The content is placed inside of the element. This content can be any HTML.

{% highlight html %}
There is content here!
{% endhighlight %}

Contextual alternatives

Like other components, easily make a panel more meaningful to a particular context by adding any of the contextual state classes.

Use the the contextual-class attribute with the following values:

  • primary
  • success
  • info
  • warning
  • danger

Default Panel. No contextual class.

Contextual Panel. Contextual class, primary.

Contextual Panel. Contextual class, success.
