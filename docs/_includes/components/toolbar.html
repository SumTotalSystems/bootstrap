Right Aligned Buttons

Toolbar elements default to the left side of the toolbar. By wrapping the elements in a div with class pull-right they will be placed on the right side of the toolbar.

Right Aligned Buttons Only

In the example above buttons are placed both on the left and right portions of the toolbar. If using only right aligned buttons this will cause the toolbar to collapse because all of the content in the toolbar would be floated to the right. Bootstrap/Foundation provides an easy fix for this. Using the clearfix class (described here in more detail) on the toolbar will prevent this from happening.
