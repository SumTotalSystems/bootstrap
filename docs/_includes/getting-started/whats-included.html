What's included

Foundation Bootstrap consists of four easy-to-consume files.

Everything that's required

Please note that everything which is required by Foundation is included in the following files. Including other scripts or style documents may cause conflicts. This site serves as a defacto standard for how all controls should appear. Please insure that if any other scripts or styles are included that they do not interfere with the Foundation base.

Precompiled Bootstrap

Once installed through the NodeJS module you should have something similar to the following:

{% highlight bash %} foundation-bootstrap/ ├── styles/ │ ├── foundation.css │ └── vendor.css └── scripts/ ├── foundation.js └── vendor.js {% endhighlight %}

This is the most basic and suggested form of Foundation: it is designed to be used in nearly any web project. Our fonts are currently imported via an include from a CDN.

Foundation Bootstrap source code

The Foundation Bootstrap source code repository includes source Less, JavaScript, and example pages. More specifically, it includes the following and more:

{% highlight bash %} bootstrap/ ├── examples/ ├── gulp/ ├── js/ │ └── vendor/ ├── less/ │ ├── angular-strap/ │ ├── bootstrap/ │ └── sumtotal/ ├── modules/ ├── templates/ └── tests/ {% endhighlight %}

The less/ and modules/ are the source code for our CSS and JS (AngularJS controls), respectively. The examples/ folder contains developer examples of Foundation usage. Beyond that, any other included file provides support for packages, license information, and development.
