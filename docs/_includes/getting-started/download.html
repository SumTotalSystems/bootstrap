Install with NodeJS

Install SumTotal Module

To download and install Foundation you'll need to clone the sumtotal-module from the TFS Git repository. After cloning you'll run Node's npm command and install it globally. The final argument in the second command below is the directory where the git clone placed the code. It will default to sumtotal-module.

{% highlight bash %}git clone http://team2:8080/tfs/DefaultCollection/Foundation/_git/sumtotal-module npm install -g sumtotal-module {% endhighlight %}

Initialize Foundation

Now that the SumTotal module is installed. The Foundation components can be applied to your project. Open a Node command prompt in your project's root. Running the sumtotal init will ask a series of questions about where your scripts and styles are stored. Answer the questions and a file called .sumtotal will be created. This should be stored in your source control system.

{% highlight bash %} sumtotal init {% endhighlight %}

Install Foundation

{% highlight bash %} sumtotal install {% endhighlight %}

Upgrading

Upgrade SumTotal Module

When upgrading the SumTotal module

{% highlight bash %}git pull origin master npm install -g . --force {% endhighlight %}

Upgrade Foundation

When upgrading the SumTotal module

{% highlight bash %}sumtotal upgrade{% endhighlight %}
