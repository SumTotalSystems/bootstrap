AngularJS

AngularJS lets you extend HTML vocabulary for your application.

AngularJS is an open-source JavaScript framework, maintained by Google, that assists with running single-page applications. Its goal is to augment web-based applications with model–view–controller (MVC) capability, in an effort to make both development and testing easier.

Other frameworks deal with HTML’s shortcomings by either abstracting away HTML, CSS, and/or JavaScript or by providing an imperative way for manipulating the DOM. Neither of these address the root problem that HTML was not designed for dynamic views.

AngularJS is a toolset for building the framework most suited to your application development. It is fully extensible and works well with other libraries. Every feature can be modified or replaced to suit your unique development workflow and feature needs. Read on to find out how.

Overview

Much like Foundation Bootstrap is a toolbox for building pages, AngularJS is a toolbox to build dynamic page views. In fact, SumTotal Foundation Bootstrap uses AngularJS for building its controls. We've standardized on using AngularJS because we feel it provides a powerful and flexible framework for building the client side interface. The beauty of Foundation is that if you choose to not use AngularJS, but use JQuery you're OK. Obviously you won't be able to take advantage of the growing number of controls that the Foundation team is building, but you're not tied to strictly using AngularJS for building the front end.

We could speak in metaphors all day long and not actually describe anything about AngularJS! To better understand how AngularJS works read the conceptual overview. Once you've seen an example application we believe you'll find yourself head over heals about all the new possibilities for client side development.
